\chapter{Missing Values, Duplikat, dan Outlier Dasar}

\begin{subcpmk}
  \item \textbf{Sub-CPMK-P3:} Menangani missing, duplikat, dan outlier sederhana dengan narasi keputusan (RPS Mg.\ 3).
\end{subcpmk}

\SesiRPS{3}{P3}{$\sim$170 menit}{CPMK-P2}

\noindent Cleaning data adalah serangkaian \textbf{keputusan yang dapat diaudit}: setiap penghapusan baris, imputasi, atau pemotongan outlier harus dapat dijelaskan kepada rekan atau pemangku kepentingan tanpa menyembunyikan dampaknya pada distribusi kelas atau variabel.

\section{Kemampuan akhir sesi}

Melaporkan NA per kolom; memilih strategi imputasi atau penghapusan; menghapus duplikat; mengidentifikasi outlier visual (boxplot) dan mendokumentasikan tindakan.

Kemampuan akhir ini mendukung CPMK-P2: Anda tidak hanya ``memperbaiki'' angka, tetapi menunjukkan bahwa Anda memahami risiko bias---misalnya imputasi median yang meratakan varians atau penghapusan baris yang mengorbankan kelompok minoritas.

\section{Teori}

\begin{konsep}
\textit{Missing} dapat MCAR/MAR/MNAR---pada praktikum, yang penting Anda \textbf{transparan}: berapa banyak yang hilang, apa yang Anda lakukan, dan risiko bias. Duplikat baris dapat merusak statistik dan model; outlier bukan selalu error---bisa sinyal---tetapi harus diputuskan dengan konteks domain.
\end{konsep}

\begin{catatan}
Contoh memakai \texttt{titanic} dari Seaborn (sama dengan Minggu 2--5) agar alur praktikum konsisten. Dokumentasikan setiap keputusan (mis.\ mengapa kolom \texttt{deck} dihapus) di Markdown.
\end{catatan}

Distribusi setelah imputasi atau penghapusan harus dibandingkan secara visual dengan data mentah minimal pada satu atau dua variabel kunci; ini membantu mendeteksi distorsi yang tidak disengaja sebelum data masuk ke model.

\section{K3 / etika}

Jangan menghapus baris berisi kelompok minoritas tanpa analisis dampak terhadap keadilan model (catatan untuk proyek akhir).

Pada dataset yang mengandung variabel sensitif (gender, asal, dll.), penghapusan selektif dapat memperburuk representasi kelompok tertentu. Laporkan proporsi kelas sebelum dan sesudah cleaning bila target bersifat sosial atau perilaku.

\section{Sarana}

\texttt{pandas}, \texttt{numpy}, \texttt{seaborn}, \texttt{matplotlib.pyplot} untuk boxplot.

\section{Prosedur kerja}

Prosedur minggu ini mengalir dari audit kuantitatif (\texttt{isna}) ke tindakan terdokumentasi dan verifikasi visual. Menyimpan \texttt{df\_raw} memungkinkan Anda mengembalikan keputusan jika asisten atau dosen mempertanyakan langkah tertentu.

\begin{enumerate}[leftmargin=*]
  \item Hitung \texttt{isna().sum()} per kolom; simpan salinan \texttt{df\_raw}.
  \item Terapkan strategi (imputasi median/mode atau \texttt{dropna} selektif) dengan komentar Markdown.
  \item \texttt{drop\_duplicates()}; laporkan berapa baris yang hilang.
  \item Plot boxplot satu kolom numerik; tulis keputusan outlier.
  \item Simpan \texttt{df\_clean} untuk minggu berikutnya.
\end{enumerate}

\section{Contoh kode}

Kode berikut menggambarkan pola kerja dari audit missing menuju \texttt{df\_clean} yang siap divisualisasikan; sesuaikan strategi dengan domain dataset Anda.

\noindent\textbf{Mengapa blok ini?} \texttt{.copy()} mencegah aliasing berbahaya saat Anda mengubah \texttt{df}; \texttt{isna().sum()} menghitung NA per kolom, \texttt{sort\_values(ascending=False)} menempatkan kolom paling banyak missing di atas.

\begin{lstlisting}[caption=Muat Titanic dan audit missing values]
import pandas as pd
import numpy as np  # opsional: operasi numerik lanjutan
import seaborn as sns

df = sns.load_dataset("titanic").copy()  # salinan independen untuk manipulasi
df_raw = df.copy()  # cadangan audit (disarankan disimpan di notebook Anda)

missing = df.isna().sum().sort_values(ascending=False)  # jumlah NA per kolom, urut menurun
print(missing[missing > 0])  # hanya kolom yang punya missing
\end{lstlisting}

\noindent\textbf{Mengapa blok ini?} Kolom \texttt{deck} sangat sparse sehingga sering dihapus; \texttt{fillna(median)} untuk \texttt{age} robust terhadap outlier; \texttt{mode()} mengambil kategori paling sering untuk \texttt{embarked}.

\begin{lstlisting}[caption=Hapus kolom sangat sparse; imputasi terarah]
df_clean = df.copy()
# deck: terlalu banyak NA untuk contoh sederhana
df_clean = df_clean.drop(columns=["deck"], errors="ignore")  # errors="ignore": aman jika kolom sudah tidak ada

df_clean["age"] = df_clean["age"].fillna(df_clean["age"].median())  # imputasi numerik: median kolom yang sama
emb_mode = df_clean["embarked"].mode()  # Series modus (bisa kosong)
if len(emb_mode) > 0:
    df_clean["embarked"] = df_clean["embarked"].fillna(emb_mode[0])  # isi NA dengan kategori tersering
\end{lstlisting}

\noindent\textbf{Mengapa blok ini?} \texttt{select\_dtypes(include="number")} memilih kolom numerik; loop mengisi sisa NA dengan median per kolom (contoh agresif---sesuaikan dengan domain).

\begin{lstlisting}[caption=Imputasi median untuk kolom numerik lain (opsional)]
num_cols = df_clean.select_dtypes(include="number").columns  # Index nama kolom numerik
for c in num_cols:
    med = df_clean[c].median()  # median per kolom (abaikan NA saat hitung)
    df_clean[c] = df_clean[c].fillna(med)
\end{lstlisting}

\noindent\textbf{Mengapa blok ini?} \texttt{drop\_duplicates()} menghapus baris identik penuh; bandingkan \texttt{shape} sebelum/sesudah untuk dokumentasi.

\begin{lstlisting}[caption=Duplikat dan shape]
print("sebelum", df_clean.shape)
df_clean = df_clean.drop_duplicates()  # hapus duplikat baris penuh
print("sesudah", df_clean.shape)
\end{lstlisting}

\noindent\textbf{Mengapa blok ini?} Boxplot memvisualkan median, kuartil, dan titik luar (\textit{outlier}) per kategori \texttt{class} pada \texttt{fare}---berguna sebelum memutuskan winsorize/log atau membiarkan model yang robust.

\begin{lstlisting}[caption=Boxplot tarif per kelas (outlier visual)]
import matplotlib.pyplot as plt
import seaborn as sns

plt.figure(figsize=(8, 4))  # ukuran figur dalam inci (lebar, tinggi)
sns.boxplot(data=df_clean, x="class", y="fare")  # perbandingan distribusi fare antar kelas kabin
plt.title("Sebaran fare per class (deteksi outlier)")
plt.tight_layout()  # rapatkan margin agar label tidak terpotong
plt.show()
\end{lstlisting}
