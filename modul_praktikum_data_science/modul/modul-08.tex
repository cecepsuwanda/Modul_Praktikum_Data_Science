\noindent Membandingkan beberapa jenis klasifikator pada data yang sama melatih intuisi \textbf{bias-variance} dan asumsi geometris (pembatas linear, jarak, atau partisi sumbu). Matriks konfusi mengubah satu angka akurasi menjadi pola salah prediksi yang bisa ditindaklanjuti.

\section{Kemampuan akhir sesi}

Membangun pipeline prapemrosesan sederhana; melatih klasifikasi; membandingkan akurasi atau F1; menampilkan \texttt{confusion\_matrix}.

Selain akurasi, pertimbangkan distribusi kelas: pada data tidak seimbang, model yang selalu memprediksi kelas mayoritas bisa tampil ``tinggi'' secara akurasi namun gagal menangkap kelas minoritas yang penting secara bisnis.

\section{Teori}

\begin{konsep}
Klasifikasi memprediksi kelas diskrit. Regresi logistik memodelkan log-odds linear; K-NN berbasis jarak; pohon keputusan memecah ruang fitur secara hierarkis. Untuk kelas tidak seimbang, akurasi bisa menyesatkan---pertimbangkan F1 per kelas.
\end{konsep}

Regresi logistik dengan regularisasi mengandalkan skala fitur yang wajar. K-NN sangat sensitif terhadap skala (jarak Euclidean). Pohon lebih toleran pada monotonik fitur numerik, tetapi tetap mudah \textit{overfit} tanpa batas kedalaman atau pemangkasan.

\section{Contoh kode}

\MulaiPenjelasanKode Memuat Iris sebagai array NumPy lalu split 75/25 dengan \texttt{stratify=y} agar tiga kelas bunga tetap proporsional di train dan test. \SelesaiPenjelasanKode

\begin{lstlisting}[caption=Data dan split (Iris)]
from sklearn.datasets import load_iris
from sklearn.model_selection import train_test_split
X, y = load_iris(return_X_y=True)  # fitur 4 kolom, label 0/1/2
X_train, X_test, y_train, y_test = train_test_split(
    X, y, test_size=0.25, random_state=42, stratify=y  # 25% data ke test; stratify menjaga proporsi kelas
)
\end{lstlisting}

\MulaiPenjelasanKode Regresi logistik sebagai model linier probabilistik; \texttt{.score} pada klasifikasi = akurasi. \SelesaiPenjelasanKode

\begin{lstlisting}[caption=LogisticRegression]
from sklearn.linear_model import LogisticRegression
log_reg = LogisticRegression(max_iter=200)  # solver butuh cukup iterasi untuk konvergen multi-kelas
log_reg.fit(X_train, y_train)
print("logreg score", log_reg.score(X_test, y_test))
\end{lstlisting}

\MulaiPenjelasanKode KNN membandingkan jarak ke-\texttt{n\_neighbors} tetangga (skala fitur penting); pohon keputusan mempartisi ruang fitur---\texttt{max\_depth} membatasi kompleksitas mengurangi overfit. \SelesaiPenjelasanKode

\begin{lstlisting}[caption=KNeighborsClassifier dan DecisionTree]
from sklearn.neighbors import KNeighborsClassifier
from sklearn.tree import DecisionTreeClassifier

knn = KNeighborsClassifier(n_neighbors=5)  # voting mayoritas di antara 5 tetangga terdekat
knn.fit(X_train, y_train)  # menyimpan data train (lazy learning)
tree = DecisionTreeClassifier(max_depth=3, random_state=42)  # kedalaman maksimum 3; random_state: pohon dapat diulang
tree.fit(X_train, y_train)
print("knn", knn.score(X_test, y_test))
print("tree", tree.score(X_test, y_test))
\end{lstlisting}

\MulaiPenjelasanKode Matriks kebingungan menghitung TP/TN/FP/FN per kelas (di Iris 3x3); \texttt{ConfusionMatrixDisplay} memplot heatmap numerik. \SelesaiPenjelasanKode

\begin{lstlisting}[caption=Confusion matrix]
from sklearn.metrics import confusion_matrix, ConfusionMatrixDisplay
import matplotlib.pyplot as plt
pred = log_reg.predict(X_test)  # label prediksi LogReg pada test
cm = confusion_matrix(y_test, pred)  # matriks frekuensi: baris=true, kolom=pred
disp = ConfusionMatrixDisplay(cm)  # bungkus untuk API plot sklearn
disp.plot()
plt.title("Confusion matrix (LogReg)")
plt.show()
\end{lstlisting}
