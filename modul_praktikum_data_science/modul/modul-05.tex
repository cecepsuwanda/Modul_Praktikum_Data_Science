\SesiRPS{5}{P4}{$\sim$170 menit}{CPMK-P2}

\noindent EDA (\textit{Exploratory Data Analysis}) mengulang siklus: merumuskan pertanyaan, memplot, lalu memeriksa ringkasan tabel. Tujuan utamanya bukan estetika, melainkan \textbf{mendukung keputusan} tentang transformasi data dan jenis model yang realistis.

\section{Kemampuan akhir sesi}

Menghasilkan histogram/KDE, scatter atau boxplot kategori, heatmap korelasi; setiap plot memiliki judul dan label sumbu; merumuskan satu hipotesis pemodelan.

Setiap visual sebaiknya diakhiri dengan satu kalimat interpretasi yang dapat dipahami rekan non-statistik: misalnya ``kelas pertama cenderung membayar tarif lebih tinggi dan memiliki proporsi survival berbeda'' alih-alih hanya menyebut nama plot.

\section{Teori}

\begin{konsep}
EDA menjawab ``apa yang data ceritakan'' sebelum asumsi model. Visual yang baik memiliki \textbf{judul}, \textbf{label sumbu}, dan \textbf{skala} yang terbaca; hindari chart junk berlebihan.
\end{konsep}

\begin{catatan}
Contoh memakai dataset \texttt{titanic} Seaborn; target biner \texttt{survived} cocok untuk hipotesis klasifikasi (mis.\ hubungan \texttt{fare}/\texttt{class} dengan survival).
\end{catatan}

Membandingkan distribusi variabel terhadap kelas target (\texttt{hue}) sering mengungkap interaksi yang tidak terlihat pada agregat tunggal; ini menjadi bahan diskusi penting sebelum memilih metrik evaluasi.

\section{Sarana}

\texttt{matplotlib.pyplot}, \texttt{seaborn}, \texttt{pandas}.

\section{Contoh kode}

Contoh berikut mendemonstrasikan empat keluarga visual pada dataset yang sama; di notebook, tambahkan sel Markdown antara plot untuk menjelaskan temuan.

\MulaiPenjelasanKode \texttt{plt.subplots(1,2)} menyusun dua sumbu sejajar. Di sumbu kiri, \texttt{histplot(..., kde=True)} menunjukkan distribusi dan kepadatan; di kanan, \texttt{scatterplot(..., hue=)} mewarnai titik menurut kelas target. \SelesaiPenjelasanKode

\begin{lstlisting}[caption=Histogram umur dan scatter umur vs fare]
import matplotlib.pyplot as plt  # antarmuka pyplot: figur dan sumbu
import seaborn as sns

df = sns.load_dataset("titanic")

fig, ax = plt.subplots(1, 2, figsize=(10, 4))  # 1 baris 2 kolom sumbu; figsize inci
sns.histplot(df["age"].dropna(), kde=True, ax=ax[0])  # dropna: histogram tanpa NA; kde: kurva halus
ax[0].set_title("Distribusi umur penumpang")
ax[0].set_xlabel("age")
sns.scatterplot(
    data=df, x="age", y="fare", hue="survived", alpha=0.65, ax=ax[1]  # hue: warna per kelas survived; alpha: transparansi
)
ax[1].set_title("Umur vs tarif (hue: survived)")
plt.tight_layout()
plt.show()
\end{lstlisting}

\MulaiPenjelasanKode Boxplot membandingkan distribusi \texttt{fare} antar \texttt{class} (median, IQR, outlier). \SelesaiPenjelasanKode

\begin{lstlisting}[caption=Boxplot tarif per kelas kabin]
plt.figure(figsize=(8, 4))  # figur baru default axes
sns.boxplot(data=df, x="class", y="fare")  # perbandingan kategori sumbu-x vs numerik sumbu-y
plt.title("Sebaran fare per class")
plt.xlabel("class")
plt.ylabel("fare")
plt.tight_layout()
plt.show()
\end{lstlisting}

\MulaiPenjelasanKode Matriks korelasi Pearson (\texttt{.corr()}) pada subset numerik divisualkan sebagai heatmap; \texttt{center=0} menekankan korelasi positif/negatif. \SelesaiPenjelasanKode

\begin{lstlisting}[caption=Heatmap korelasi fitur numerik]
num = df[["survived", "pclass", "age", "sibsp", "parch", "fare"]].dropna()  # hanya baris lengkap untuk korelasi
plt.figure(figsize=(7, 5))
sns.heatmap(num.corr(), annot=True, fmt=".2f", cmap="vlag", center=0)  # annot: tulis angka di sel; fmt: dua desimal
plt.title("Korelasi antar variabel numerik")
plt.tight_layout()
plt.show()
\end{lstlisting}

\MulaiPenjelasanKode \texttt{pairplot} menampilkan scatter per pasangan fitur; \texttt{corner=True} hanya segitiga bawah (mengurangi redundansi); \texttt{hue} memisahkan warna menurut \texttt{survived}. \SelesaiPenjelasanKode

\begin{lstlisting}[caption=Pairplot subset (dengan hue survival)]
cols = ["age", "fare", "pclass"]
sns.pairplot(
    df[cols + ["survived"]].dropna(),
    hue="survived",  # legenda warna per kelas
    corner=True,  # panel duplikat di atas diagonal dihilangkan
    plot_kws={"alpha": 0.6},  # transparansi titik scatter
)
plt.show()
\end{lstlisting}
