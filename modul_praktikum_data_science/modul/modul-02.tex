\section{Teori dan bahan kajian}

Memuat data adalah gerbang pertama ke seluruh pipeline analitik. Pandas menyatukan berbagai sumber (berkas lokal, URL, pustaka dataset) ke dalam objek \texttt{DataFrame} yang dapat diinspeksi secara seragam.

\begin{konsep}
\texttt{DataFrame} adalah tabel dengan indeks baris dan nama kolom. \texttt{dtypes} membedakan \texttt{int64}, \texttt{float64}, \texttt{object} (sering berarti teks atau campuran). Fungsi \texttt{info()} memberi ringkasan non-null dan tipe; \texttt{describe()} merangkum statistik numerik secara default.
\end{konsep}

\begin{catatan}
Path berkas: gunakan path relatif ke folder kerja notebook atau unggah ke Colab dengan \texttt{files.upload()}. Dokumentasikan path di Markdown agar asisten dapat mereproduksi.
\end{catatan}

\begin{catatan}
Contoh di bawah memakai dataset bawaan Seaborn (\texttt{tips}, \texttt{titanic}) untuk latihan tanpa unduhan manual. Dataset terbuka lain: bandingkan dengan \texttt{fetch\_openml}. Jika Anda memakai data sendiri, cantumkan URL dan lisensi di notebook.
\end{catatan}

\section{Contoh kode}

Contoh berikut menunjukkan variasi sumber data tanpa mengganti pola inspeksi: selalu mulai dari \texttt{head}, \texttt{info}, lalu \texttt{describe} bila kolom numerik sudah masuk akal.

\begin{lstlisting}[caption=Dua dataset Seaborn: tips dan Titanic]
import pandas as pd
import seaborn as sns

df_tips = sns.load_dataset("tips")
df_titanic = sns.load_dataset("titanic")

print("tips", df_tips.shape, df_tips.dtypes.head())
print("titanic", df_titanic.shape)
print(df_titanic.head())
\end{lstlisting}

\begin{lstlisting}[caption=Ringkasan struktur dan statistik (Titanic)]
print(df_titanic.info())
print(df_titanic.describe(include="all").T.head(14))
\end{lstlisting}

\begin{lstlisting}[caption=Alternatif sumber terbuka: OpenML (perlu jaringan pertama kali)]
from sklearn.datasets import fetch_openml

iris_oml = fetch_openml(name="iris", version=1, as_frame=True)
df_iris = iris_oml.frame
print(df_iris.shape, df_iris.columns.tolist())
print(df_iris.head())
\end{lstlisting}

\begin{lstlisting}[caption=Membaca CSV dari URL (contoh format terbuka)]
url = (
    "https://raw.githubusercontent.com/datasciencedojo/datasets/master/titanic.csv"
)
df_url = pd.read_csv(url)
print(df_url.shape)
\end{lstlisting}

\begin{lstlisting}[caption={Membaca Excel (satu sheet, jika ada berkas lokal)}]
df_xlsx = pd.read_excel("data/contoh.xlsx", sheet_name=0)
print(df_xlsx.dtypes.value_counts())
\end{lstlisting}

\begin{lstlisting}[caption=Membandingkan dua kerangka data]
def ringkas(nama, frame):
    return {
        "nama": nama,
        "baris": len(frame),
        "kolom": frame.shape[1],
        "missing_total": int(frame.isna().sum().sum()),
    }

print(ringkas("tips", df_tips))
print(ringkas("titanic", df_titanic))
\end{lstlisting}
