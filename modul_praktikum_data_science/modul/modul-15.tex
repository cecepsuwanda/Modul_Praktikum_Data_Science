\section{Lingkup proyek}

\begin{konsep}
Proyek mini membuktikan integrasi capaian P2--P4. Topik disetujui dosen: masalah jelas, data legal, minimal satu model, metrik selaras jenis target, kesimpulan non-teknis.
\end{konsep}

Pilih dataset yang ukuran dan kompleksitasnya realistis untuk waktu lab; lebih baik analisis yang rapi pada data sedang daripada model rumit tanpa dokumentasi. Prioritas draft pertama adalah alur utuh, bukan skor tertinggi.

\MulaiPenjelasanKode \texttt{Pipeline} mengikat prapemroses dan estimator agar \texttt{fit} pada train menerapkan transform yang sama saat \texttt{predict} pada data baru---wajib untuk proyek akhir yang dinilai ulang. \SelesaiPenjelasanKode

\begin{lstlisting}[caption=Pipeline disarankan]
from sklearn.pipeline import Pipeline
from sklearn.preprocessing import StandardScaler
from sklearn.linear_model import LogisticRegression

model = Pipeline([
    ("sc", StandardScaler()),  # nama langkah bebas; dipakai di GridSearchCV sebagai prefiks
    ("clf", LogisticRegression(max_iter=1000)),
])
