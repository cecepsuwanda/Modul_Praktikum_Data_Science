\chapter{Klastering: K-Means dan Silhouette}

\begin{subcpmk}
  \item \textbf{Sub-CPMK-P8:} K-Means pada fitur numerik ter-skala; argumen pemilihan $k$ dengan silhouette atau inertia (RPS Mg.\ 10).
\end{subcpmk}

\SesiRPS{10}{P8}{$\sim$170 menit}{CPMK-P3}

\noindent Klastering mencari struktur tanpa label; hasilnya bergantung pada skala fitur dan pemilihan $k$. Oleh karena itu, standarisasi (\texttt{StandardScaler}) dan kurva diagnostik (inertia, silhouette) wajib dibahas bersama narasi, bukan hanya angka tunggal.

\section{Kemampuan akhir sesi}

Menjalankan K-Means pada data numerik ter-skala; membandingkan beberapa $k$ dengan silhouette dan/atau inertia; mendeskripsikan profil rata-rata tiap klaster untuk komunikasi hasil.

Profil klaster (\texttt{groupby}) membantu menamai segmen secara domain-driven, misalnya ``klaster dengan pengeluaran tinggi dan frekuensi rendah''---meskipun penamaan tetap subjektif dan perlu validasi bisnis.

\section{Teori}

\begin{konsep}
Klastering \textit{unsupervised}: tidak ada label. K-Means meminimalkan inertia dalam cluster; hasil dipengaruhi skala fitur---gunakan \texttt{StandardScaler}.
\end{konsep}

K-Means mengasumsikan bentuk klaster relatif bulat dan ukuran sebanding; untuk data sangat tidak simetris, algoritma lain (DBSCAN, GMM) mungkin lebih cocok, tetapi di luar cakupan minimal praktikum ini.

\section{Contoh kode}

\begin{lstlisting}[caption=Data dan scaling]
from sklearn.datasets import make_blobs
from sklearn.preprocessing import StandardScaler

X, _ = make_blobs(n_samples=400, centers=4, cluster_std=0.6, random_state=42)
scaler = StandardScaler()
Xs = scaler.fit_transform(X)
\end{lstlisting}

\begin{lstlisting}[caption=KMeans dan inertia]
from sklearn.cluster import KMeans
km = KMeans(n_clusters=4, random_state=42, n_init="auto")
labels = km.fit_predict(Xs)
print("inertia", km.inertia_)
\end{lstlisting}

\begin{lstlisting}[caption=Silhouette score untuk beberapa k]
from sklearn.metrics import silhouette_score
for k in range(2, 8):
    kmk = KMeans(n_clusters=k, random_state=42, n_init="auto")
    lab = kmk.fit_predict(Xs)
    s = silhouette_score(Xs, lab)
    print(k, s)
\end{lstlisting}

\begin{lstlisting}[caption=Profil cluster (rata-rata per fitur)]
import numpy as np
import pandas as pd
dfp = pd.DataFrame(Xs)
dfp["cluster"] = labels
print(dfp.groupby("cluster").mean())
\end{lstlisting}

\section{Alternatif: data nyata (Iris)}

\begin{catatan}
\texttt{make\_blobs} mengontrol bentuk klaster sintetis. Untuk membandingkan, dataset bawaan Iris (empat fitur numerik) sering membentuk pola yang cocok dengan $k=3$---ingat klaster tidak sama dengan label spesies; jangan memakai \texttt{target} sebagai input K-Means saat mensimulasikan unsupervised.
\end{catatan}

Membandingkan label klaster dengan label asli (\textit{adjusted rand index}, dll.) boleh sebagai latihan tambahan, tetapi fokus praktikum tetap pada deskripsi pola tanpa ``kunci jawaban'' supervisi.

\begin{lstlisting}[caption={K-Means pada Iris (fitur saja, di-skala)}]
from sklearn.datasets import load_iris
from sklearn.cluster import KMeans
from sklearn.preprocessing import StandardScaler
from sklearn.metrics import silhouette_score

iris = load_iris(as_frame=True)
X_iris = iris.data.values
scaler_i = StandardScaler()
Xs_i = scaler_i.fit_transform(X_iris)

km_i = KMeans(n_clusters=3, random_state=42, n_init="auto")
lab_i = km_i.fit_predict(Xs_i)
print("inertia", km_i.inertia_)
print("silhouette k=3", silhouette_score(Xs_i, lab_i))
\end{lstlisting}
