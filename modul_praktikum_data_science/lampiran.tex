\section*{D. Panduan laporan ringkas}

\begin{enumerate}[leftmargin=*]
  \item Halaman judul: nama, NIM, judul, MK, tanggal.
  \item Ringkasan 150--250 kata.
  \item Data dan metode; hasil (tabel + gambar); keterbatasan.
  \item Referensi dengan sitasi sumber data.
\end{enumerate}

\section*{E. Template penamaan}

\begin{itemize}[leftmargin=*]
  \item Notebook mingguan: \texttt{NIM\_TIFxxxPrak\_M\textit{nn}.ipynb}
  \item Notebook latihan di folder \texttt{notebooks/}: \texttt{minggu\_\textit{nn}.ipynb}
  \item \texttt{requirements.txt} atau cuplikan versi paket untuk reproduktifitas.
\end{itemize}

\section*{F. Lembar peer review inti}

Tujuan jelas; langkah dapat diikuti; bagian membingungkan; visual terbaik; minimal dua saran perbaikan.

\section*{G. Lembar observasi asisten}

Checklist: K3; Jupyter mandiri; sumber data; prosedur utama selesai; siap lanjut atau perlu remedial.

\section*{H. Glosarium singkat}

\begin{description}[style=nextline, widest=Pipeline:, itemsep=0.2em]
  \item[CRISP-DM:] Siklus proyek data dari pemahaman bisnis hingga deployment.
  \item[EDA:] Eksplorasi data sebelum pemodelan formal.
  \item[Leakage:] Informasi target bocor ke fitur latih.
  \item[Pipeline:] Rangkaian \texttt{transform} + model di \sklearn{}.
\end{description}
